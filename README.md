# Sales_Data_Analysis

## Introduction
Analyzing Sales data using python to prepare data then create a database using SQL Server 2019 Management Studio, using SQL Server Integration Services (SSIS) for creating automated process, using Microsoft Analysis Services(SSAS) and MDX to create an advanced report then finally creating dashboard with Power BI Desktop to highlight the findings.
